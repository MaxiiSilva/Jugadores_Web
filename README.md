Trabajo Práctico Entregable – Diseño Web

Este proyecto consiste en el desarrollo de una tabla de registros de usuarios utilizando diseño y desarrollo web. La aplicación administrá usuarios mediante operaciones básicas de gestión.

Funcionalidades:

➕ Agregar usuarios

❌ Eliminar usuarios

🗑️ Borrar registros

📋 Mostrar usuarios en una tabla dinámica
